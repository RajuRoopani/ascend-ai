## 🎯 What is Ascend?

Ascend is a career intelligence platform for software engineers targeting elite companies. It combines live job data from 31 top companies with deep AI-powered interview preparation — all personalized to *your specific background*.

### The Problem
Senior SWE interviews at FAANG+ and AI startups are brutal. Generic LeetCode grind and YouTube system design videos don't cut it. You need:
- **Current** openings, not stale job boards
- **Company-specific** prep (Meta ≠ Google ≠ Anthropic)
- **Level-calibrated** depth (L5 vs L6 require fundamentally different answers)
- **Personalized** advice that knows your background

### The Solution
Ascend fetches live roles every 6 hours from Greenhouse, Lever, and Ashby ATSs, then uses Claude Opus to generate deeply personalized prep plans per role — and uniquely, per *your resume*.

## ✨ Core Features

### 🔍 Live Job Board
- **445+ active SWE roles** from 31 companies, refreshed every 6 hours
- Filter by level (L3–L7+), remote, company tier, and full-text search
- Direct links to apply + instant access to AI prep per role

### ✦ Career Match (Resume Intelligence)
The flagship feature:
1. **Paste your resume** → Ascend AI parses your profile (level, skills, experience, strengths)
2. **Match scoring** → Every active role is scored against your background (0–100)
3. **Personalized prep** → Generate prep plans that reference *your* specific experience

Match scores reflect:
- Skills overlap with job requirements (40%)
- Level alignment (30%)
- Role/domain relevance (20%)
- Growth opportunity (10%)

### 📝 5 AI Prep Plan Types (per job)
All generated by Claude Opus, then cached for instant re-access:

| Tab | What you get |
|-----|-------------|
| 📝 **Coding Plan** | 8–10 DSA patterns, 6-week LeetCode roadmap, 15 must-do problems with company context |
| 🏗 **System Design** | Level-calibrated depth, 5 company-specific design questions, 30-min mock |
| 🎯 **Behavioral** | 8 STAR story templates, all 16 Amazon LPs mapped, bar raiser playbook |
| 🏢 **Company Tips** | Exact loop structure, TC ranges by level, cultural signals, red flags |
| ⚡ **Edge Tech** | Top-of-band skills for AI era (LLM infra, eBPF, distributed systems), L→L+1 promo criteria |

### ✦ Personalized Prep (Resume-Aware)
When you've uploaded your resume, every prep plan becomes hyper-personalized:
- References your actual past companies and projects
- Highlights what you already know vs. gaps to close
- Tailors study plans to your current level and target level
- Gives you "candidate X coming from Google interviewing at Anthropic" specific advice

## 🚀 Getting Started

### Prerequisites
- Docker & Docker Compose
- Anthropic API key (`sk-ant-...`)

### Quick Start

```bash

# Configure environment
```

## 📡 API Reference

```
GET  /api/jobs                          # Paginated job list (filter: q, level, remote, company, tier)
GET  /api/jobs/{id}                     # Job detail
GET  /api/jobs/{id}/prep/{type}         # Get/generate cached prep plan
POST /api/jobs/{id}/prep/{type}/personalized  # Resume-personalized prep plan
GET  /api/companies                     # All companies grouped by tier
GET  /api/companies/{slug}              # Company profile + open roles
POST /api/resume/analyze                # Parse resume → profile + job matches
POST /api/refresh                       # Trigger manual job refresh
GET  /health                            # Health check
```

**Prep types:** `coding` · `system_design` · `behavioral` · `company_tips` · `edge_tech`

## 💡 How the AI Works

### Resume Intelligence Pipeline
```
Resume text
    ↓ Claude Haiku (fast, cheap)
Profile extraction: level, skills, companies, strengths, headline
    ↓ Claude Haiku
Score all 445 active roles → top 20 with match%, match_reasons, skill_gaps
    ↓ Result
Personalized job matches displayed with score rings
```

### Prep Plan Generation
```
Job (title + level + description + company)
    + optionally: your resume context
    ↓ Claude Opus (5 parallel generation paths)
Detailed markdown: DSA plan / System Design / Behavioral / Company Tips / Edge Tech
    ↓ PostgreSQL cache (UNIQUE job_id + content_type)
Instant return on subsequent requests
```

### Cost Model
- Prep plan generation: ~$0.20–0.40 per plan (Claude Opus, cached after first load)
- Resume analysis: ~$0.002 (Claude Haiku for parsing + matching)
- Personalized prep: ~$0.30–0.50 per plan (Claude Opus, not cached)

## 🎨 Design Philosophy

Ascend's UI is built on a "terminal meets luxury" dark design system:

- **Background:** `#07080C` near-black with surface layers (`#0D0F18`, `#131621`)
- **Accent:** Indigo/violet gradient (`#6366F1 → #8B5CF6`) with glow on hover
- **Typography:** Geist Sans (body) + Geist Mono (numbers, badges, code) — Vercel's premium open-source font family
- **Motion:** Shimmer skeletons, radial gradient hero glow, gradient headline text, live-dot pulse

Inspired by Linear's radical minimalism, Levels.fyi's data density, and the editorial warmth of YC Jobs.

## 📊 Database Schema

Three core tables:
- **`companies`** — 31 companies with ATS config, interview loop, TC ranges
- **`jobs`** — Active SWE roles with FTS GIN index, level detection, dedup by (company_id, external_id)
- **`prep_plans`** — Cached AI-generated content (UNIQUE per job + content_type), with token/cost tracking
